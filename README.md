# Kemet VR - Ancient Egypt Virtual Experience 🐪✨

Kemet VR is a multi-page web application designed to provide an immersive educational experience about Ancient Egypt. Visitors can explore the history of Pharaohs, the pantheon of Gods, the art of mummification, and the great architectural monuments.

## 🎯 Project Overview
This project was built as a core HTML/CSS web development assignment with strict constraints. The goal was to build a complete, fully functional, and well-designed website relying **only** on pure HTML5 and CSS3.

**Key Requirements Met:**
- **10 Unique HTML Pages** linked together seamlessly.
- **Zero JavaScript:** All interactivity and navigation rely purely on HTML/CSS.
- **No CSS Frameworks:** 100% custom styling (No Bootstrap, Tailwind, etc.).
- **HTML Elements Used:** Semantic tags, Data Tables, Forms, and Iframes (Google Maps & YouTube).
- **Modern Layouts:** Extensively uses CSS Flexbox and CSS Grid for responsive design.

## 📁 Project Structure (10 Pages)
1. `index.html` - Home / Hero Section
2. `pharaohs.html` - Famous Rulers (Data Table)
3. `gods.html` - The Pantheon (CSS Grid Cards)
4. `mummies.html` - Mummification Process (Ordered/Unordered Lists)
5. `gallery.html` - Artifacts Gallery (Image Grid)
6. `tour.html` - Book a Virtual Tour (Complex HTML Form)
7. `map.html` - Locations Map (Google Maps Iframe)
8. `videos.html` - Documentaries (YouTube Iframe)
9. `architecture.html` - The Great Architecture (Flexbox Layout)
10. `about.html` - About the Team & Project Details

## 🛠️ Technologies Used
- **HTML5** (Semantic structuring, Forms, Tables, Iframes)
- **CSS3** (Custom variables, Flexbox, Grid, Hover effects, Smooth scrolling)
